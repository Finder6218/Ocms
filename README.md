# 离退休干部管理系统

《离退休干部管理系统》是管理离退休干部的一款软件,方便查询,一键导出数据。 

系统包含：   
1、信息管理：离退休干部统计表、逝世离退休干部统计表   
2、信息查询：信息查询 

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe(或Hadp.exe)"文件即可完整运行对应的信息管理系统。
